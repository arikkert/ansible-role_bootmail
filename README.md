Role Name
=========

Send email to root account on shutdown and bootup

Requirements
------------

root email account is configured for this use .e.g. by a proper alias in /etc/aliases thats forwards it to a sysadmin mbox

Role Variables
--------------

None

Dependencies
------------

None

----------------
